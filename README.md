# Bomber

Простой смс-бомбер на запросах для тестирования защищенности номера телефона. Он не обходит защиту в виде токенов и тд. Это скоро добавится.

## Структура проекта

```
.
├── main.py                  # Главная точка входа
├── proxy_manager.py         # Модуль управления прокси
├── proxies.txt              # Список URL прокси
├── services/
│   └── service_manager.py   # Определение сервисов и обработчики
└── venv/                    # Виртуальное окружение
```

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/aeemao/Bomber
cd Bomber
```

2. Создайте виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Конфигурация

### Прокси

Добавьте URL прокси в `proxies.txt`, по одному в строке:
```
http://proxy1.com:8080
http://proxy2.com:8080
http://proxy3.com:8080
```
Бесплатные рабочие прокси можно получить с помощью моего чекера: 
https://github.com/aeemao/PCHECKER

## Использование

Запустите основной скрипт:
```bash
python3 main.py
```

При появлении запроса введите номер телефона в формате: `79999999999` (Без плюса!)

Скрипт будет:
1. Загружать прокси из `proxies.txt`
2. Создавать параллельные задачи для каждой комбинации сервиса и прокси
3. Отправлять SMS-запросы через настроенные прокси
4. Отображать результаты успеха (✅) и ошибок (❌)
